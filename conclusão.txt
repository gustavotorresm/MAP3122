O presente trabalho nos permitiu conhecer melhor o processo de resolver um problema modelado por uma equação diferencial com métodos numéricos. Para o problema escolhido ,foi necessário implementar o método dos trapézios e um algoritmo para splinescúbicos.  O algoritmo que implementou o método dos trapézios teve sua rectidão verificada pelo Método das Soluções Manufaturadas, pois foi possível constatar que a ordem do algoritmo era 2, como é previsto pela teoria. Para verificar o algoritmo dos splines cúbicos, comparou-se os coeficientes fornecidos por ele com os de exemplos conhecidos [4]. Além disso, observou-se que as cúbicas obtidas tinham comportamento muito semelhante as curvas obtidas pelo método dos trapézios.
O problema da curva de perseguição mostrou-se muito interessante para o estudo de métodos numéricos. O problema genérico é descrito por relações muito simples e possui muitos parâmetros livres, a saber, a curva descrita pelo perseguido e a constante de proporcionalidade entre as velocidades do perseguidor e do perseguido. Dessa forma, depois de implementar os algoritmos, foi possível testá-los para inúmeros casos e verificar o comportamento da curva de perseguição para diferentes relações de velocidade. Para alguns casos, verificou-se também qual seria a relação necessária para o encontro dos dois.
